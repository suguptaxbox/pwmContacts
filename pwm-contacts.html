Understanding Your Appliances

SIEMENS Dishwasher

Error Code E15
ISSUE: water that was collected in the centre / bottom so the system shows E15 error.
DETAILS/SOLUTION: Clean it and then do a hard reboot of the washer
Error Code E0
ISSUE: indicates a fault in the heating system, usually a problem with the heating element or temperature sensor.
DETAILS/SOLUTION: To fix this, it is recommended to first close the water tap and switch off the appliance for safety. You can try cleaning the filters and checking for blockages, but if the error persists, you will need to contact a professional for a diagnosis and repair.

SIEMENS Washing Machine

Door Does not open
ISSUE: Child Lock may have been activated.
DETAILS/SOLUTION: Press and hold the start button for 3 to 5 seconds. If it is a Child Lock issue then it should unlock .
Error Code F23
ISSUE: water has been detected in the base of the washing machine, triggering the anti-flood or AquaStop system.
DETAILS/SOLUTION: detected water in its base, which activates the anti-flood system. To troubleshoot, first unplug the machine and close the water tap to prevent further water damage. Then, you need to locate and remove the water from the base tray, which often requires tilting the machine or removing the rear panel. Inspect the drain pump, filters, and hoses for blockages or leaks, and if you can't resolve the issue, you should contact professional service.
